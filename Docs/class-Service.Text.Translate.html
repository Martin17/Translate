# __translateToGoogle( string $stringToTranslate )

Method to translate with de Google Service.

Method to translate with de Google Service.

Ok, Google Translation service is not free anymore... But we can connect to the online free service with this little trick! The text is divided into phrases to translate one by one. This is due to free online API responds with a multidimensional array with the options of translation of each sentence, making it difficult to generate the text again.
